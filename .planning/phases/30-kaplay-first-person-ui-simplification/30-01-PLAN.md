# Phase 30-01 Plan: Kaplay First-Person UI Simplification

## Objective

Simplify the Kaplay first-person interface per user feedback:
- Remove the confusing "Quick View" group label; rename to "Utility"
- Remove Feed and Status tabs; show only Actions
- Default to grid mode instead of first-person
- Preserve mode switching ([1] first-person, [2] grid)

## Context

- First-person mode is the same game as grid, different perspective; future work will add more storytelling to first-person
- Shared UI primitives (addButton, addPanel, etc.) remain in `packages/kaplay-demo/src/shared.ts`
- `buildActionGroups` from engine presenter drives action display in both modes

## Files

- `packages/engine/src/escape-the-dungeon/ui/presenter.ts` — rename Quick View → Utility

- `packages/kaplay-demo/src/main.ts` — default to gridNavigation
- `docs-site/lib/escape-the-dungeon/ui/presenter.ts` — sync Quick View → Utility (if used by tests)
