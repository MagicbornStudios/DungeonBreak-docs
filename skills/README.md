# Escape the Dungeon Skills

Skills for AI assistants (Cursor, Codex, Claude) to run game simulation.

## escape-the-dungeon-simulation

Run Escape the Dungeon conceptually in chat. One player message = one turn.

**Codex:** Install from this repo:
```bash
# If you have the skill-installer
scripts/install-skill-from-github.py --repo <owner>/<repo> --path skills/escape-the-dungeon-simulation
```

**Cursor:** Rule at `.cursor/rules/escape-the-dungeon-simulation.mdc` (applies when `scratch/**` is open).

**Claude:** See `.claude/escape-the-dungeon-simulation.md`.
